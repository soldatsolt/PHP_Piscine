<- Vertex ----------------------------------------------------------------------
Данный класс хранит в себе однородный координаты точки (homogeneous coordinates)
(x,y,z,w) и цвет точки. Если цвет не был задан, класс делает точку белой.
Если координаты x, y, z не были заданы корректно (коррдинаты следует подавать как
один массив - array( 'x' => 1.0, 'y' => 0.0, 'z' => 0.0)), класс задаст конструкт
задаст коррдинаты (0,0,0). То же относится и к координате w.
Общий вид правильного инициирования объекта выглядит следующим образом:
$unit = new Vertex(array('x' => *, 'y' => *, 'z' => *, 'color' => объект) );
